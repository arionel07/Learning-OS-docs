## Обзор

Learnin OS построен на трёх основных слоях:

- **Ядро** — Среда выполнения, которая загружает и управляет модулями
- **Контент** — Файлы Markdown и MDX, составляющие документацию
- **UI** — Frontend на Next.js, который отображает всё

## Слой ядра

Ядро отвечает за:

- Обнаружение и загрузку модулей
- Разрешение wiki-ссылок между страницами
- Построение дерева навигации
- Генерацию поисковых индексов

```typescript
import { createOS } from '@learningos/core'

const os = createOS({
  contentDir: './src/content',
  modules: ['javascript', 'typescript'],
})
```

## Слой контента

Весь контент находится в `src/content/docs/`, организованный по языкам:

```
src/content/docs/
  en/
    index.md
    getting-started/
      installation.md
  ru/
    index.md
```

Каждый файл поддерживает frontmatter для метаданных:

```yaml
---
title: Моя страница
description: Краткое описание
order: 1
tags: [пример, руководство]
---
```

## Слой UI

UI — это приложение Next.js с:

- Файловой маршрутизацией через `app/docs/[lang]/[[...slug]]`
- Серверной загрузкой данных
- Клиентской интерактивностью (поиск, тема, боковая панель)
- Рендерингом MDX с пользовательскими компонентами

## Поток данных

```
Файловая система → docs.lib.ts → page.tsx → DocPage.tsx → Браузер
```

1. `docs.lib.ts` читает `.md` и `.mdx` файлы
2. `page.tsx` получает нужный документ на стороне сервера
3. `DocPage.tsx` рендерит контент на стороне клиента
4. Браузер отображает итоговый HTML
